# MapPi3 V1.2.6 HTTPS Phone GPS Preview

## Purpose

V1.2.6 creates an HTTPS preview path so the phone browser can request location permission for Daily Exercise.

## Live preview

Current tunnel URL:

```text
https://mappi3-cak3d.loca.lt/
```

This forwards to the VM dev server on port `8080`.

## Run commands

In one terminal/process, keep the app running:

```bash
npm run dev -- --host 0.0.0.0 --port 8080
```

In another terminal/process, keep the HTTPS tunnel running:

```bash
npm run tunnel
```

## Phone test steps

1. Open `https://mappi3-cak3d.loca.lt/` on the phone.
2. Go to **Daily Exercise**.
3. Tap **Start walk**.
4. Approve location permission if prompted.
5. Walk outside/near a window for better GPS lock.
6. Watch the Leaflet/OpenStreetMap trace.

## Known limitation

Localtunnel is a temporary preview tool. If it restarts, the URL may fail or need restarting. It is good enough for immediate HTTPS/GPS testing, but not the final deployment path.

## Better permanent options later

- Vercel HTTPS deployment.
- Tailscale Serve HTTPS.
