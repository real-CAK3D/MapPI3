# MapPi3 V1.2.15 Route Pack Editor State

## Added

- Plan shows dirty/saved route-pack status.
- Save route pack creates or updates a named local custom route pack without mutating the seed route.
- Duplicate route creates a custom copy of the selected route.
- Custom routes persist in `mappi3.routes` and show alongside seed route cards.
- JSON route-pack export includes route geometry, built-in waypoints, custom waypoints, POIs, metadata, and app version.
- JSON route-pack import accepts pasted MapPi3 JSON and adds it as a local custom route.

## Storage

- `mappi3.routes` stores custom route pack cards/routes.
- `mappi3.customWaypointsByRoute` stores active waypoint edits per route.

## Safety

Route packs are planning aids. Imported/custom route data can be stale or wrong; verify important water/shelter/danger markers with field-safe navigation sources.
