# MapPi3 V1.0.4 Notes

## Purpose

V1.0.4 introduces the first real route-pack data model under the prototype UI.

## Added

- `src/data/routePacks.json` sample route-pack catalog.
- `src/data/routePacks.js` mapper for UI cards and raw route-pack metadata.
- UI now imports route cards, primary waypoints, map preview path, distance, elevation, storage estimate, and GPX route name from route-pack data.
- Local storage migration keeps downloaded state while refreshing route metadata from the current seed catalog.
- `docs/route-pack-format.md` documents the route-pack schema.

## Route-pack fields now modeled

- route identity and discovery fields
- difficulty, route type, tags
- distance/time/elevation/storage estimate
- sync state placeholder
- offline MBTiles bounds/layers/zoom metadata
- GeoJSON-style route geometry
- temporary SVG map preview path
- waypoints and POIs

## Still mocked

- Real map rendering from MBTiles.
- Real route-pack downloader.
- Real route-pack validation.
- Real Supabase route catalog.
- Real GPX import parser.
