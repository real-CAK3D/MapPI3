# MapPi3 V1.2.14 Waypoint Polish

## Added

- Custom waypoints can be edited after creation.
- Custom waypoint markers are draggable on the Leaflet map.
- Snap-to-trail toggle controls whether dropped/dragged waypoints are projected back to the selected route.
- Custom icon picker for waypoint markers.
- Selected route GPX export now includes custom waypoints, including notes as GPX descriptions.
- Navigate's next waypoint logic now considers built-in and custom waypoints sorted by mile marker.

## Safety / accuracy

Waypoint placement is local browser state. Trail snap depends on route-pack geometry and is for planning assistance only; carry real navigation and verify critical water/shelter/danger markers in the field.
