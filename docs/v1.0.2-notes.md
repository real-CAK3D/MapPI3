# MapPi3 V1.0.2 Notes

## Purpose

V1.0.2 turns the V1.0.1 static concept screen into a more app-like phone prototype.

## Added

- Interactive section tabs: Explore, Plan, Navigate, Saved, Field Kit, Survival, Settings.
- Download-to-Pi-SD mock state: route cards can switch to Downloaded.
- Custom route builder mock with tools for drawing, waypoints, POIs, and GPX import/export.
- Active hike dashboard with route stats, GPS accuracy, heading, sunset, and offline pack status.
- Saved offline packs panel to reinforce modular SD-card storage.
- Sense HAT Field Kit panel with calibration, bag mode, and sensor status.
- Survival quick cards with safety-honest language.
- Settings panel with connection mode, units, map theme, and future integration badges.
- Richer terrain/topo-style map mock with contours, grid, POIs, elevation card, and pulsing GPS marker.

## Still mocked

- Real map engine / MBTiles.
- Real GPS feed.
- Real route search/geocoding.
- Real GPX parsing.
- Real Sense HAT access.
- Real Supabase/Vercel/GitHub remote wiring.
