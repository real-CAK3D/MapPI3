# MapPi3 V1.2.28 Near Me Radius + Completion

## Added

- `near me` now ranks by distance from phone GPS/current origin, falling back to Lewiston/Sabbatus local home.
- Search radius selector: 25, 80, 150, 300, all.
- Search results filter by radius unless set to all, preventing Kentucky/remote routes from showing in Maine-local browsing.
- Drilldown cards now collapse: click an area to hide other areas; click it again to reopen. Same for mountain/cluster cards.
- Route cards show distance from current/home origin.
- Navigate includes a route completion bar and Mark trail complete action.
- Completed trails are saved locally now and displayed in Saved; Supabase can later sync the same records to profile stats.

## Accuracy note

Distance ranking is straight-line-to-trailhead until road routing is wired. Completion is local/browser storage until Supabase login/profile support exists.
