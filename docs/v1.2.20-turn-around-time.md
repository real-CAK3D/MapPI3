# MapPi3 V1.2.20 Turn-Around Time

## Added

- Launch now estimates a turn-around time from the selected route duration.
- Default turn-around is about 45% of planned route time, capped by a conservative 6:30 PM daylight/safety buffer.
- Editable turn-around time field.
- Turn-around checklist row.
- Share/export hike plan includes turn-around and daylight/safety buffer.
- Readiness log stores turn-around time.
- Navigate toolbar shows a `turn <time>` badge when Launch has a turn-around time.

## Safety

The daylight buffer is a simple conservative placeholder, not live sunset/terrain math. Adjust for actual local sunset, season, weather, route difficulty, crew pace, and turnaround rules.
