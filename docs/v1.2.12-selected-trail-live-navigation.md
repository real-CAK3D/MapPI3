# MapPi3 V1.2.12 Selected Trail Live Navigation

## Added

- Navigate now renders the selected route's real geometry on the Leaflet map instead of only the stylized mock route.
- Selected route waypoints render as map markers.
- Start Route uses phone GPS when the page is a secure Tailscale HTTPS origin.
- If phone GPS is unavailable, navigation clearly falls back to a simulator that follows the selected route geometry.
- Live/offline status shows distance from selected trail corridor, current route progress, next waypoint, distance remaining, and off-route warning above 250 ft.
- Route Detail now exports GPX for the selected route.

## Accuracy boundary

This is accurate to the selected route-pack geometry and the phone GPS accuracy reported by the browser. It is not a substitute for a real map/compass/emergency tool. Current off-route distance is measured to the closest route vertex in the local route pack; a later pass should upgrade that to closest segment projection for smoother corridor math.
