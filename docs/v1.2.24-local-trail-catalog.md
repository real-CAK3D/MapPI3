# MapPi3 V1.2.24 Local Trail Catalog Expansion

## Why

The search bar was only showing a tiny local seed catalog: Hocking Hills plus two demo routes. That made search feel stuck even though the filtering code worked.

## Added

- Expanded `src/data/routePacks.json` from 3 routes to 23 local seed route packs.
- Added Maine/New England-heavy routes plus nearby regional examples.
- Added searchable `region` to mapped route cards.
- Kept every route as local seed data; no Supabase dependency yet.
- Updated pack count to use the real route count instead of a hardcoded denominator.

## Notes

These are planning seed entries with approximate coordinates/geometry. Supabase can replace or enrich the catalog later with verified public route data, trailhead metadata, road travel estimates, and shared route packs.
