# MapPi3 V1.2.26 Grouped Route Search

## Added

- Ranked search results instead of flat filtering.
- Exact trail/name/tag matches come first.
- Related routes from the same `mountainArea`, `region`, or `place` are pulled in after the exact match.
- Explore results are grouped into:
  - Best trail matches
  - Same mountain / location area
  - Walking paths / easier local options
  - Other related route packs
- Route cards now show land/type labels such as:
  - Public land
  - Appalachian Trail
  - Walking path
  - Public preserve
  - Public trail / verify access
- Route data includes `landType`, `catalogCategory`, and `mountainArea` metadata.

## Example

Searching `table rock` should put Table Rock routes first, then show the surrounding Grafton Notch / Route 26 routes below it.

## Safety

Land/type labels are planning aids from seed data, not legal access guarantees. Verify public access, trail status, parking, and current conditions before hiking.
