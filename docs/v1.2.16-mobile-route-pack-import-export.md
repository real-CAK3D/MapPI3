# MapPi3 V1.2.16 Mobile Route Pack Import/Export

## Added

- Replaced paste-prompt JSON route import with a mobile-friendly file picker.
- Validates `.mappi3.json` / JSON route-pack files before import.
- Shows an import preview with route name, place, mileage, built-in waypoint count, and custom waypoint count.
- Import requires confirmation via **Import preview**.
- Saved tab now supports route-pack actions:
  - Open route in Plan
  - Export route GPX
  - Export MapPi3 JSON pack
  - Delete custom route packs
- Delete is limited to local custom route packs; seed route packs are preserved.

## Validation

Importer accepts MapPi3 route-pack schema:

```text
mappi3-route-pack-v1
```

or a direct route object with `name` and `geometry.coordinates`. Invalid files show a friendly blocked/import error in the Plan import panel.

## Safety

Imported/custom route data can be stale or wrong. Critical water, shelter, danger, and navigation details must still be verified with field-safe sources.
