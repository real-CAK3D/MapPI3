# MapPi3 V1.2.18 Pre-Hike Launch Checklist

## Added

- New **Launch** tab between Plan and Navigate.
- Select route before starting navigation.
- Pre-hike checklist covers:
  - offline pack freshness
  - HTTPS/GPS origin
  - weather checked
  - water
  - food/calories
  - first aid/emergency gear
  - battery/power bank
- Start navigation from the checklist.
- Saves timestamped readiness logs locally in `mappi3.readinessLog`.
- Checklist state persists locally in `mappi3.launchChecklist`.

## Safety

The checklist is a field-readiness aid, not a guarantee. MapPi3 does not replace backup navigation, local trail guidance, emergency gear, or judgment.
