# MapPi3 V1.2.17 Offline Pack Status

## Added

- Offline pack status card in Route Detail.
- Freshness labels: today / Nd old / not cached, with stale after 14 days.
- Refresh and delete offline controls for route packs.
- Saved tab now shows local vs live/catalog source labels.
- Ready-for-hike checklist covering offline data, HTTPS GPS origin, weather, waypoints, and backup safety.
- Download/refresh stamps `offlineSavedAt` in local route state.

## Safety

Offline status means MapPi3 has local browser route-pack state. It does not guarantee trail conditions, closures, water safety, or emergency readiness. Carry backup navigation and field gear.
