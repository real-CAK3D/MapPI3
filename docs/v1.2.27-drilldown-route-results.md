# MapPi3 V1.2.27 Drilldown Route Results

## Added

- Removed search-suggestion chips from below the top search bar.
- Quick filters are now only: `near me`, `day hike`, and `overnight`.
- Explore uses compact drilldown cards:
  1. Area cards
  2. Mountain / trail-cluster cards
  3. Trail cards under the selected cluster
- Ranking still keeps closest trail/name/tag matches first inside the card hierarchy.
- Route detail now includes a Leaflet map for the selected route, with route geometry and waypoint markers.
- Route detail explicitly labels geometry honesty: local seed geometry is approximate until verified GPX/Supabase trail geometry is attached.

## Notes

This prevents huge search result pages while keeping nearby trails discoverable. For true trail accuracy, the next data step is verified GPX/OSM/Supabase geometry, not hand-drawn approximations.
