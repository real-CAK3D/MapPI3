# MapPi3 V1.2.7 Tunnel White Screen Fix

## Problem

The Localtunnel HTTPS page could show the app shell but then load a white screen on the phone. Investigation showed Vite returned `403 Forbidden` for `/src/main.jsx` through the tunnel host.

## Cause

Vite's dev server host protection did not allow the Localtunnel host:

```text
mappi3-cak3d.loca.lt
```

So the HTML loaded, but the JavaScript module was blocked.

## Fix

Added `vite.config.js` with explicit allowed hosts for:

- `mappi3-cak3d.loca.lt`
- `localhost`
- `127.0.0.1`
- `150.136.165.61`
- `100.82.165.23`

Restart the Vite dev server after this change.

## Localtunnel password

If Localtunnel shows the anti-abuse page asking for an IP/password, use the VM public IP:

```text
150.136.165.61
```

Then the app should load normally.
