# MapPi3 V1.2.10 Route Details + Open Food/Drink Entry

## Added

- Route cards now open a detail panel with summary, stats, tags, waypoints, POIs/safety notes, offline layer format, Start route, and Download/Refresh pack actions.
- Route search now also scans route summaries, waypoint names/types, and POI labels/types.
- Start route selects the route, jumps to Navigate, resets progress, and starts the navigation timer.
- Food/beverage catalog now includes unhealthy foods and drinks such as soda, milk, milkshakes, candy, chips, burgers, pizza, fries, donuts, etc.
- Food and beverage inputs now support arbitrary typed entries:
  - typed food adds a 150 calorie estimate
  - typed drink adds a 12 oz estimate

## Intent

The health tracker is not meant to judge the hiker. It should capture real intake — good, bad, or messy — so the hiker can see where they are and improve over time.
