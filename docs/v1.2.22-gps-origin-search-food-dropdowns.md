# MapPi3 V1.2.22 GPS Origin, Search Detail, and Food Dropdowns

## Added/fixed

- Launch can use phone GPS as the travel-origin point for trailhead distance estimates.
- Origin point persists in `mappi3.originPoint`; falls back to local/home reference if GPS is unavailable.
- Search detail panel no longer stays stuck on Hocking Hills when filtered results do not include the old selected route.
- Food and beverage suggestions now behave as dropdowns: hidden until typing, cleared after selection.
- Tabs grid updated for the added Launch tab.

## Limits

Travel distance is still straight-line plus rough drive-time speed until real road routing is wired. Phone GPS requires the HTTPS Tailscale URL and browser permission.
