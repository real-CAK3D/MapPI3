# MapPi3 V1.0.3 Notes

## Purpose

V1.0.3 makes the prototype feel alive and local-first instead of only navigational screens.

## Added

- Browser-persisted state with `localStorage` for active tab, settings, downloaded route status, elapsed time, and route progress.
- Simulated moving GPS marker along the mock route.
- Start/Pause recording and Reset controls in Navigate.
- Route progress line and live tracked mileage tied to the simulator.
- GPX export button that downloads a real sample `mappi3-sample-route.gpx` file from the current sample waypoints.
- GPX import mock state that documents the future parsing path.
- Saved packs are now derived from downloaded route state.
- Settings include a reset-local-state control for testing field-device behavior.

## Still mocked

- Real GPSD feed.
- Real MBTiles map engine.
- Real GPX file picker/parser.
- Real route-pack download worker.
- Real Sense HAT service.
