# MapPi3 V1.2.11 Health Daily Editable Intake

## Added

- Editable intake rows: rename food/drink entries, adjust calories, or delete them.
- Today snapshot: calories in, calories burned, water logged, water target, hydration status, and net calories.
- Save today: stores a local daily summary.
- Save + reset intake: stores today then clears food list and water ounces for a fresh day.
- Daily history: shows recent local summaries with calories, water, item count, and hydration note.

## Notes

This stays local to the phone/browser origin for now. It is intentionally honest and nonjudgmental: the hiker can log soda, milk, candy, burgers, or anything custom, then use trends to improve later.
