# MapPi3 V1.2.13 Route Segment Accuracy + Waypoint Placement

## Added

- Navigation off-route/progress math now projects phone location to the closest route segment instead of only checking route vertices.
- Plan tab now uses the selected trail and shows it on the Leaflet map.
- Custom waypoint placement:
  - choose waypoint type/name/notes
  - tap **Place on map**, then tap the map
  - or quick-place at 25%, 50%, or 75% of the selected trail
  - custom waypoints show as pink markers and persist per route in localStorage
  - custom waypoints can be deleted

## Storage

Custom waypoints are local to the phone/browser origin under:

```text
mappi3.customWaypointsByRoute
```

## Accuracy boundary

Segment projection gives smoother progress/off-route estimates than nearest vertex. It still depends on the quality of the route-pack geometry and phone GPS accuracy.
